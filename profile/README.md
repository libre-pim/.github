# **Librepim - LTS Fork of Akeneo PIM Community Edition**

**Librepim** is an open-source, community-driven fork of the **Akeneo PIM Community Edition**. It provides **Long-Term Support (LTS)**, security patches, bug fixes, and performance improvements, ensuring that businesses and developers can continue to use a stable, secure Product Information Management (PIM) system while the community drives its evolution.

Our mission is to keep the **Akeneo PIM Community Edition** alive, secure, and up-to-date for all users, while fostering contributions from the community. **Librepim** is for those who wish to continue using and improving the Community Edition without migrating to Akeneo’s other offerings.

---

## Key Features

* **Long-Term Support (LTS)**: Continuous security patches, bug fixes, and performance improvements.
* **Seamless Migration**: Migrate from Akeneo PIM Community Edition to **Librepim** with minimal effort.
* **Community-Driven**: Contributions are always welcome! Help improve **Librepim** and make it better for everyone.
* **Secure & Stable**: Focused on maintaining a secure and stable platform for product data management.

## Getting Started

### Installation Instructions

To install **Librepim**, follow the same installation instructions for **Akeneo PIM Community Edition**, with slight adjustments for **Librepim**-specific configurations:

[**Librepim Installation Guide**](https://docs.akeneo.com/master/install_pim/index.html)

For local development using Docker, you can build the Docker image:

```bash
docker build --target dev -t librepim/pim-php-dev:master .
```

## About Librepim

**Librepim** is a community-driven, open-source fork of **Akeneo PIM Community Edition**.
